在开发移动端的项目时，移动端适配是开发第一个要面对的问题。

## viewport
在移动端中，我们经常可以看到head标签中这段代码：
```bash
<meta name='viewport' content='width=device-width,initial-scale=1,user-scale=no' />
```
通过meta标签对viewport的设置，定义了页面的缩放比例，页面宽度与视口宽度一致
